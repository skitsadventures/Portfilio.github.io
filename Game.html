Drone System

I've used this in a previous project as an anti-aircraft turret, but it can easily be used for various other purposes.


  • Autonomous roaming within a defined area, with random target selection, smooth movement forces, rotation alignment, and slowdown near targets

  • Hover mechanics with dynamic height adjustment, oscillation effects, and physics-based lift/damping to maintain a smooth floating behavior

  • Propeller-based balancing with torque and damping to keep the drone upright and stable while allowing controlled leaning into movement

  • Use of Unity physics (Rigidbody, AddForce, AddTorque) for realistic motion instead of purely kinematic movement

  • Collision avoidance using horizontal cylindrical detection and physics-based impulses to prevent collisions with obstacles

  • Debugging and visualization tools including Gizmos and Debug.DrawLine to monitor drone behavior in the editor

  • Integration between systems so that roaming, hovering, and avoidance work together seamlessly
